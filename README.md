# Recife Pet

## Sobre: 
Aplicação feita usando a base Sakai PrimeNG de uma loja de PET com todas as operações do CRUD e gráficos representativos com informações relevanntes para os clientes.

## Tecnologias:
- React (TypeScript)
- Firebase
- SCSS
- Vercel
